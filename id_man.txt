DESCRIPTION       Print user and group information for each specified USER, or (
                  when USER omitted) for the current user.

VERSION           id (GNU coreutils) 8.32

RELATED COMMANDS  aa,acpid,add,amdgpu,amidi,aplaymidi,arecordmidi,ati

EXAMPLE           >id
                  uid=1000(jehad) gid=1000(jehad) groups=1000(jehad)
